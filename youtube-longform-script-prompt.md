# YouTube Long-Form Script Prompt

## Purpose
Creates structured YouTube video scripts for women’s empowerment topics.

## Prompt
Create a YouTube video script for the topic: [insert topic].

Include:
- Strong hook
- Story or relatable example
- Spiritual or psychological insight
- Key lessons
- Call to action

## Example

Input:
Topic: Stop overgiving in relationships

Output:
Hook: “The fastest way to lose yourself is by giving too much to people who haven’t earned it…”
